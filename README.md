# PataAmiga
Pets Adoption sistem 


  To create the database in your Microsoft SQL Server verify if the ConnectionsString in appsettings.jsop is valid in your instance then 
open Tools -> Nuget Packege Maneger -> Packege Maneger Console and run this commands:

Add-Migration "Nome da Migracao"   // to create the Migration
Update-Database                    // to install the migration in your database 

To run the project Build -> Build the solution then press F5 to Run, a localhost instance should be open in your browser
